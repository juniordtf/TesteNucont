A solução dos exercícios 1, 2, 3 e 4 estão nos arquivos “Transform_a”, “Transform_b”, “Transform_c” e “Transform_d”, respectivamente. Para salvar os resultados no MongoDB é preciso realizar os seguintes passos abaixo:1. Abrir um terminal na pasta “nucont” e executar o comando “npm install” para instalar as dependências do projeto;2. Nesse mesmo terminal executar o comando “meteor” para subir o mongoDB;3. Executar o comando “node index.js” para de fato salvar os resultados no mongoDB.
